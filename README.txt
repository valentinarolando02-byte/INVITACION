Instrucciones rápidas para publicar en GitHub Pages:

1) Crear un nuevo repositorio en GitHub (ej. 'invitacion-townhall').
2) Subir todos los archivos de este zip (index.html y invitacion.png) al repositorio.
   - En la web: Repo -> Add file -> Upload files -> Commit.
3) Ir a Settings -> Pages (o Settings -> Code and automation -> Pages).
4) En 'Source' elegir branch 'main' (o 'master') y carpeta '/ (root)'. Guardar.
5) Esperar unos segundos/minutos; la URL será: https://<tu_usuario>.github.io/<nombre-repo>/
   (ej: https://valentinarolando.github.io/invitacion-townhall/)
6) Probá abrir la URL; debería mostrar SOLO la imagen (index.html la carga).
7) Usá esa URL directa en tu Adaptive Card (campo Image.url). Ej:
   https://<tu_usuario>.github.io/<nombre-repo>/invitacion.png

Si querés, te doy el JSON listo para la Adaptive Card una vez tengas la URL.
